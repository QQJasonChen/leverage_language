Privacy Policy for YouGlish Extension

Last updated: [Date]

Data Collection

This extension collects and stores the following data locally on your device:

  • Your selected language preferences
  • API keys you provide for translation services
  • Search history and saved phrases
  • Audio recordings you choose to save

Data Usage

The data is used to:

  • Provide language learning features
  • Enable translation services
  • Save your learning progress locally

Data Storage

All data is stored locally using Chrome's storage API. No data is sent to our servers.

Third-Party Services

This extension may connect to:

  • YouGlish.com for pronunciation examples
  • YouTube for video content
  • Translation APIs (when you provide API keys)
  • Notion API (if you choose to export to Notion)
